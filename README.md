# NDA

This repository holds the Non-Disclosure Agreement (NDA) that must be signed before being granted access to any private repositories, communications, data, or other confidential material belonging to **David Cahill**, the **Furbidden** GitHub organization, **Cybrix.AI**, or any other sole business venture operated by David Cahill (each, a "Covered Entity").

## If you're here to sign (start here)

1. Read **[NDA.md](NDA.md)** in full.
2. Follow **[SIGNING.md](SIGNING.md)** — a step-by-step walkthrough. Most people will use the GitHub-website-only path; no installs needed.
3. David reviews and merges your PR. That merge is the countersignature.
4. You'll be invited to the relevant org/repos shortly after.

## What's in this repo

- [NDA.md](NDA.md) — the agreement itself.
- [SIGNING.md](SIGNING.md) — step-by-step signing instructions.
- [signatures/](signatures/) — one file per signer. Where your signature gets committed.
- [signatures/TEMPLATE.md](signatures/TEMPLATE.md) — copy this to create your signature file.
- [SETUP.md](SETUP.md) — one-time setup for the repo owner. Signers can ignore this.
